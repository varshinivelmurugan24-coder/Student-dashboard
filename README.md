# Student-dashboard
Step 1: Open Excel

Open Microsoft Excel.

Create a New Workbook.

Step 2: Create a Sheet Named “README”

At the bottom, rename Sheet1 to README.

This sheet will explain the workbook.

Step 3: Add Basic Project Information

Create two columns:

A (Field)	B (Description)
File Name	Name of the Excel file
Project Name	Project title
Author	Creator name
Created Date	File creation date
Version	File version
Last Updated	Latest update date
Step 4: Describe Each Sheet

Add a table like this:

Sheet Name	Purpose	Notes
Data	Raw data storage	Do not edit manually
Dashboard	Charts and summary	Auto updated
Report	Final report output	Export to PDF
Step 5: Explain Column Meanings

Example:

Column Name	Description	Example
ID	Unique record number	1001
Name	Customer name	Ravi
Date	Transaction date	10-03-2026
Step 6: Add Instructions for Users

Example section:

Instruction	Details
Data Entry	Enter data only in the Data sheet
Do Not Edit	Formula columns should not be changed
Refresh	Click refresh to update dashboard
Step 7: Add Formatting

Use Bold headers

Use borders for tables

Use different colors for sections

Step 8: Save the File

Click File → Save As

Choose Excel Workbook (.xlsx).
